1. ¿Qué es una ecuación cuadrática?


Las ecuaciones cuadráticas o ecuaciones de segundo grado son aquellas en donde el exponente del término desconocido está elevado al cuadrado, es decir, la incógnita está elevada al exponente 2. Tienen la forma general de un trinomio:


ax2+ bx + c = 0


  • donde a, b y c son números reales, con a 0:
  • a es el coeficiente de x2.
  • b es el coeficiente de x .
  • c es el término independiente.

2. Clasificación ecuación cuadrática


Las ecuaciones cuadráticas se clasifican en dos tipos según sus coeficientes a, b y c en completas e incompletas.

  • Si los tres coeficientes son distintos de 0, la ecuación es completa.
  • Si uno o los dos coeficientes b ó c son igual a 0, la ecuación es incompleta.

Ejemplo:


Ecuación completa


2x2 + x -3 = 0; con a = 2, b = 1, c = -3 x 2 - x + 5 = 0; con a = 1, b = -1, c = 5


Ecuación incompleta


x2 - 3x = 0; con a = 1, b = 3 3x2 - 27 = 0; con a = 3, b = -27


3. Método de resolución ecuación cuadrática


Existen tres formas principales de resolver ecuaciones de segundo grado:


a. Método factorización


2x2 - 8x -4 = 3x - x2


1. El primer paso para factorizar una ecuación es transportar todos los términos a un lado de la ecuación, manteniendo positivo el término x2.


2x2 - x2- 8x - 3x - 4


Para combinar los términos, suma o resta todos los términos x2, los términos x y las constantes (términos enteros), transportándolos a un lado de la ecuación hasta que no quede nada en el otro lado. Una vez que te quedes sin términos restantes, simplemente escribe "0" en ese lado del signo igual (=).


3x2-11x-4=0


2. Dado que 3x2 solo tiene un conjunto de posibles factores, 3x y x puedes colocarlos entre paréntesis: (3x ± ?) (x ± ?)= 0. Luego, realiza un proceso de descarte para reemplazar los factores de 4 y encontrar una combinación que cuando se multiplique dé como resultado “-11x”. Puedes utilizar las combinaciones: 4 y 1, o 2 y 2, ya que, al multiplicar ambas, se obtiene 4. Solo recuerda que uno de los términos debe ser negativo, ya que el término es -4. Haz varias pruebas y empieza con esta combinación de factores: ( 3x + 1 ) ( x - 4 ) = 0. Si los multiplicas, obtendrás 3x2-12x+x-4=0 Si combinas los términos -12x+x te da, -11x que es el término medio que estamos buscando. Con esto, acabas de factorizar la ecuación.

3. Iguala cada conjunto entre paréntesis a cero como ecuaciones separadas. Al hacerlo, encontrarás dos valores para x. que harán que toda la ecuación sea igual a cero ( 3x + 1 ) ( x -4 ) = 0. Ahora que la ecuación está factorizada, todo lo que tienes que hacer es poner la expresión en cada conjunto de paréntesis igual a cero. Pero, ¿por qué? Porque para obtener cero multiplicando, tenemos el "principio, regla o propiedad" que un factor debe ser cero, entonces al menos uno de los factores entre paréntesis, como ( 3x + 1 ) ( x - 4 ) debe ser cero; ( 3x + 1 ) o bien ( x - 4 ) debe ser igual a cero. Por lo tanto, se escribe 3x+1=0 y también x - 4 = 0

4. Posibles soluciones de una ecuación cuadrática


Las ecuaciones cuadráticas pueden tener una solución real, dos soluciones reales o dos soluciones complejas conjugadas


Recuerda: Δ = b2 - 4ac


  • Si Δ > 0 entonces hay dos soluciones reales y distintas.
  • Si Δ = 0 entonces hay dos soluciones reales e iguales..
  • Si Δ < 0 entonces no tiene soluciones reales.

6. Modelamientos de problemas de la vida cotidiana


Problema 1


La longitud de un terreno rectangular es doble que el ancho. Si la longitud se aumenta en 40 m y el ancho en 6 m, el área se hace doble. Halla las dimensiones del terreno


Resolución


Sea x = el ancho del terreno y 2x = la longitud del terreno, por lo tanto, el área del terreno es 2xx = 2x2

Si se aumenta la longitud en 40 m, esta sería (2x + 40)m y aumentando el ancho en 6m, este sería (x + 6)m. El área ahora será:

( 2x + 40 )( x + 6 ) = Área del terreno

Se debe multiplicar polinomios: ( 2x2 + 12x + 40x + 240 )

Reducir términos semejantes: ( 2x2 + 52 x + 240 ) = 0

Transponiendo y reduciendo: ( x2 - 26x -120 ) = 0

Resolviendo esta ecuación se halla que x = 30 y x = -4. Aceptamos la solución x = 30


Respuesta: El ancho del terreno es 30 m y la longitud es 60 m.


Problema 2


Si es que el área de un rectángulo es 78 unidades cuadradas y su lado más largo tiene 7 unidades de longitud más que el lado más corto, ¿cuáles son las longitudes de los lados?


Resolución


El lado más corto mide x. Esto significa que el lado más largo es x + 7.

Ahora, considerando que el área de un rectángulo es encontrada al multiplicar las longitudes de sus lados, se tiene que

x ( x + 7 ) = 78

Expandiendo y escribiendo a la ecuación en la forma ax2+ bx + c = 0 tenemos:

x2 + 7x - 78 = 0

Resolviendo esta ecuación, se tiene como posibles soluciones x = -13 y x = 6

Dado que no se puede tener longitudes negativas, la solución es x = 6


Respuesta: Las longitudes del rectángulo son 6 unidades y 13 unidades respectivamente.
